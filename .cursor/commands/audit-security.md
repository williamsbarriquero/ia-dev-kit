# Auditoria de Segurança

Execute as seguintes ações para auditoria rápida de segurança:
1. Realize varredura de credenciais hardcoded.
2. Verifique dependências com vulnerabilidades conhecidas (ex: `npm audit`).
3. Aplique um checklist OWASP Top 10 simplificado.
4. Produza um Output contendo um Relatório com severidade e remediações.
